Política de Privacidade — Luminarls Última atualização: 19/08/2025 1. Introdução O Luminarls é um bot para Discord projetado para melhorar a experiência de comunidades, oferecendo funcionalidades como economia de Lucoins, exploração, sistema de empregos, tickets, AutoMod e criação de embeds. Esta Política de Privacidade explica como coletamos, usamos e armazenamos os dados dos usuários. 2. Dados coletados O Luminarls não coleta nem armazena o conteúdo das mensagens dos usuários fora do Discord. Ele coleta apenas dados necessários para o funcionamento dos recursos do bot, como: * Economia e Ranking * Saldo de Lucoins, progresso em empregos e cooldowns de exploração. * Histórico de daily e apostas para funcionalidade interna. * Tickets * Logs de tickets, transcrições em .txt ou embeds no servidor. * Configurações do Bot * Preferências de tickets, embeds salvos, palavras bloqueadas no AutoMod. * Metadados * IDs de usuário, IDs de servidor/canal, datas e horários de interações com o bot. 3. Armazenamento dos dados * Os dados são armazenados em SQLite local ou em MongoDB (quando aplicável). * Nenhum conteúdo de mensagens privadas é armazenado fora do Discord. * Dados são usados exclusivamente para: * Funcionalidades internas do bot. * Manutenção de ranking, economia e tickets. * Melhorar a experiência do usuário. 4. Compartilhamento de dados * O Luminarls não compartilha dados pessoais ou conteúdo de mensagens com terceiros. * Logs ou transcrições de tickets ficam apenas no servidor onde o ticket foi criado. 5. Controle do usuário * Os usuários não podem desativar seletivamente o rastreamento de dados internos do bot, pois ele depende de interações com comandos, ranking e tickets. * Usuários podem: * Evitar interagir com o bot. * Configurar permissões de canal para limitar a atuação do bot. * Solicitar remoção de dados entrando em contato com o desenvolvedor. 6. Segurança * Os dados são armazenados com boas práticas de segurança. * Apenas o proprietário do bot e administradores autorizados têm acesso aos dados coletados. 7. Exclusão de dados * Usuários podem solicitar que seus dados sejam removidos, enviando mensagem para o desenvolvedor do bot. * A exclusão inclui saldo, progresso, tickets e preferências, mas não inclui conteúdo de mensagens, que não é armazenado. 8.
